

这是一到动态规划的题目。如何解决此类题目呢？从整体来看，有几步：
- 定义状态变量；
  - `dp[i][j]`表示到达`i, j` 能拿到的礼物；
- 定义状态转移方程；
  - `dp[i][j] = dp[i-1][j] + dp[i][j-1] + nums[i][j]`
- 定义初始变量；
  - 考虑特例，可以从上面的状态转移方程可以看看出，可能会出现出界的情况；
  - `dp[0][0]`
  - `dp[0][j]`
  - `dp[i][0]`
- 定义遍历顺序；
  - 因为这是一个二维数组，要做的工作是填充这个二维数组，先遍历i还是先遍历j呢？
- 举例子，推导dp数组；
  - 举几个例子看一下，是否和上述推导的结果一致；
